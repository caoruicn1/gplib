Detailed Description

Squared exponential kernel with noise inference.

This kernel is defined as: sig ^ 2 * exp(- ((x - xp) * (x - xp)')/ 2 * l) + sig_noise ^ 2 * I

Note
params : vector of hyperparameters 0 : sig, 1 : l (length scale), 2 : sig_noise.
